# Wake-On-Lan

A CLI application to send Wake-On-Lan packets.

## Getting started

> dart pub global activate -s git https://github.com/fluttercandies/dart_wake_on_lan

```console
$ wol -h
-h, --help                                                                                                                                                                                                                                              
    --host                                                                                                                                                                                                                                              
    --mac
    --repeat
```
